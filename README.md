# campo-minado-Skill-Tree
🎮 Sobre o projeto  O objetivo deste projeto é transformar o clássico Campo Minado em uma experiência mais dinâmica e divertida, adicionando elementos de progressão estilo RPG.  Ao jogar, o jogador ganha moedas e experiência conforme vence partidas ou realiza ações especiais. Essas moedas podem ser utilizadas para desbloquear melhorias permanentes.
